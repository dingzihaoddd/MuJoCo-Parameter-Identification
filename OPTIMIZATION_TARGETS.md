# MuJoCo 动力学参数辨识 — 优化目标与策略

## 当前基线

| 指标 | 值 |
|------|-----|
| 辨识参数 | damping, frictionloss (2/4) |
| 优化方法 | differential_evolution |
| 激励信号 | 正弦 0.7 Hz + 5 初始角度拼接 |
| damping 误差 | 868% (真值 0.1 → 辨识 0.97) |
| frictionloss 误差 | 884% (真值 0.05 → 辨识 0.49) |
| 最终损失 | ~3.19 |

**核心问题**: damping 和 frictionloss 高度耦合，优化器将两者同时推高到接近搜索边界，说明损失地形中对角线方向存在宽阔的"伪谷底"——两个参数同比例放大时，速度符号函数和速度线性项叠加后轨迹相似，MSE 损失无法有效区分。

## 优化目标（按优先级排列）

### 1. 核心目标：2 参数辨识精度 < 10%

- damping 相对误差 < 10%
- frictionloss 相对误差 < 10%
- 正确定位真值附近 (d=0.1, f=0.05)，而非耦合谷底

### 2. 扩展目标：4 参数同时辨识

- mass, damping, frictionloss, armature 全部辨识
- 每个参数相对误差 < 15%
- 解决 mass/armature 与 damping/frictionloss 之间的跨组耦合

### 3. 鲁棒性目标

- 不同随机初始猜测下均收敛到真值（成功率 > 80%）
- 加入 1-5% 观测噪声后误差不翻倍
- 不同激励信号类型下均有效

## 策略路线图

### 阶段 A：改进损失函数（解决耦合谷底）

阻尼力矩 = -damping * qd，摩擦力矩 = -frictionloss * sign(qd)。当 qd 符号变化时两者可部分替代。

**方案 A1：加正则化惩罚耦合方向**
- 沿 "damping + frictionloss = const" 方向加惩罚项，压缩伪谷底
- 惩罚项: `λ * (damping/frictionloss_ratio - expected_ratio)^2`

**方案 A2：输出误差加权**
- 对速度符号切换时刻附近的样本加权，这些点最能区分阻尼（比例）与摩擦（符号函数）
- 权重: `w[i] = 1 + α * exp(-(qd[i])^2 / σ^2)` — 零速附近加权

**方案 A3：多损失分量拆分**
- 分别计算正速度段和负速度段的 MSE，迫使两者同时匹配
- loss = MSE(qd[qd>0]) + MSE(qd[qd<0]) + weight_q * MSE(q)

### 阶段 B：改进激励信号设计

**方案 B1：扫频信号（chirp）**
- 0.1 → 5 Hz 对数扫频，覆盖更宽频率范围
- 频率变化产生不同速度剖面，增强可辨识性

**方案 B2：分段恒速信号**
- 设计力矩使摆锤在不同恒定速度下运行
- 恒速段可精确测量 damping（比例于速度）+ frictionloss（常数）
- 线性回归分离两者

**方案 B3：bang-bang 控制**
- 最大正力矩 → 零力矩 → 最大负力矩 → 零力矩
- 加速段暴露惯性参数，减速段暴露阻尼/摩擦

### 阶段 C：改进优化策略

**方案 C1：两阶段优化**
- Stage 1: 固定摩擦=0，在高速段数据上辨识 damping
- Stage 2: 固定 damping，在低速/零速附近辨识 frictionloss
- 迭代交替直到收敛

**方案 C2：网格搜索 + 局部优化**
- 先在粗糙网格上评估损失地形全貌
- 取 top-K 低损失点分别启动 L-BFGS-B
- 可视化损失等高线，确认全局最小值位置

**方案 C3：贝叶斯优化**
- 用 Gaussian Process 代理模型替代直接仿真
- 利用采集函数平衡探索与利用
- 适合仿真开销大的场景

### 阶段 D：扩展到 4 参数

- mass 和 armature 通过多频率激励（高频测量惯性效应，低频测量重力效应）分离
- 先辨识 damping/frictionloss（用阶段 A-C 的方案），固定后辨识 mass/armature
- 或交替优化：冻结一组参数优化另一组

## 量化指标

| 目标 | 指标 | 当前值 | 目标值 |
|------|------|--------|--------|
| damping 误差 | 相对误差 % | 868% | < 10% |
| frictionloss 误差 | 相对误差 % | 884% | < 10% |
| 损失值 | MSE | 3.19 | < 1e-4 |
| 收敛成功率 | 10 次不同初始猜测 | ~0% | > 80% |
| 噪声鲁棒性 | 5% 噪声下误差增幅 | 未测 | < 2x |

## 实施优先级建议

1. **立即实施**: C2 (网格搜索+可视化) — 先看清损失地形，确认真值处是否真的是全局最小
2. **随后实施**: B1 (扫频信号) — 更丰富的激励 → 更强的可辨识性
3. **进一步实施**: A2 (零速加权) 或 C1 (两阶段优化) — 针对性地打破耦合
4. **最终目标**: D (4 参数) + C3 (贝叶斯优化)
